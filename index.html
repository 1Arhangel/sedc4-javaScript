Homework 1

Create a functional HTML / JavaScript calculator It must have:

  • a output display where the result is displayed
  • buttons for the digits (0-9)
  • buttons for the operations (+, -, *, /)
  • button for operation execution (=)

The design of the buttons and the display is not subject to evaluation - but make it look pretty if you know how :) The calculator must support doing operations on (at least) single digit numbers. The division operator must do integer division (9/4 = 2, not, 2.25) The result of the operation must be displayed in the output display - displaying the operation and operands are not required There is a referent implementation at this location
