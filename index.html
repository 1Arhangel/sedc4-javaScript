Homework 1

JavaScript calculator

You must create a functional HTML / JavaScript calculator (CSS is allowed, but not mandatory). It must have:

  • a output display where the result is displayed
  • buttons for the digits (0-9)
  • buttons for the operations (+, -, *, /)
  • button for operation execution (=)
  • optional button for clearance (C)

The design of the buttons and the display is not subject to evaluation - but make it look pretty if you know how :)

The calculator must support doing operations on (at least) single digit numbers.

The division operator must do integer division (9/4 = 2, not, 2.25)

The result of the operation must be displayed in the output display - displaying the operation and operands are not required

There is a referent implementation at this location

Note that the referent implementation is just for design overview, as the code is intentionally more complicated than neccessary.

Bonus: - Make the calculator work from the keyboard (number keys enter numbers, enter calculates, escape clears)

Homework 2

Number to words

Description:

Our user needs to print invoices for some trinkets she will be selling. She is not good at reading numbers, so we must help her. She wants to enter a number that should be invoiced, and to receive the text that should be put in the "with words" part of the invoice. You must help her print the invoices.

Implement a JavaScript function that will convert a given number to its word representation. The number can be assumed to be an JavaScript integer, in the range [1 - 999,999,999,999], but it would be nice if the function gave/threw out an error message that the input is invalid. Note that there's no need to handle negative numbers - but they should be detected as invalid input. The textual representation should be done in English - and you can test with these valid examples:

| English

--|---------- 1 | "One" 2 | "Two" 11 | "Eleven" 21 | "Twenty one" 192 | "One hundred ninety two" 1187 | "One thousand one hundred eighty seven" 2000 | "Two thousand" 21011 | "Twenty one thousand eleven" 12341678 | "Twelve million three hundred forty one thousand six hundred seventy eight" 128341679901 | "One hundred twenty eight billion three hundred forty one million six hundred seventy nine thousand nine hundred one".

The result should be returned from the function as a result - not displayed directly to the user, alerted or logged to the console. You must use the following signature of the method:

function toWords(number) {
   // your code here
   return result;
}
